from binary-tactics dir
NOTE: 1, 2, 4, 5, 7 only need to be run once.

0. install dependencies
pip install ipython cyclone==0.7 zeo httplib2
(link to cyclone: https://github.com/fiorix/cyclone/commit/4a508168540946389ac021600ecb9d5254fe2adb)

1. prepare zodb:
$ mkdir logs
$ mkdir -p DBs/Events
$ mkdir -p DBs/World
$ runzeo -C zeoEvents.conf &
$ runzeo -C zeoWorld.conf &

2. set ip in rproxy_server.
(IP setting needs to be more intelligently.)

3. start rproxy_server:
$ ipython equanimity/rproxy_server.py

4. create world inside zodb:
$ ipython equanimity/create_world.py

5. start auth_server:
$ ipython equanimity/auth_server.py

6. add users to ZODB:
(verify that equanimity/equanimity_client.py is pointing to the correct addr/IP)
$ ipython equanimity/create_players.py

7. grant the users Fields and prepare the battlequeue:
$ ipython equanimity/setup_zeo_battle.py

8. run the battle server:
$ ipython equanimity/battle_server.py '(0, 0)'

9. get the JS sources:
$ cd web; sh get_js.sh

10. http://<host>:8888/auth/login
u: atkr, p: atkr or u: defndr, p: defndr

11. http://<host>:8888/battle/static/jsonrpc.html
